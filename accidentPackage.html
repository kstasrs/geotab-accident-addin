Accident Data Package

Comprehensive incident evidence & diagnostic report generator

How to Use This Tool ▶
1
Enter search criteria

Type a vehicle name or driver name (auto-complete will suggest matches). Set the date/time of the incident. You can leave fields blank — the tool discovers related data automatically.

2
Generate the package

Click Generate Package. The tool collects GPS logs, trips, safety events, accelerometer/collision data, fault codes, HOS/ELD, DVIR inspections, diagnostics, messages, and camera footage in one batch.

3
Review & download

Browse results in the tabbed report. Download everything as an Excel workbook (multi-sheet) or CSV ZIP for upload to claim systems. Use the Media tab to download dash-cam videos.

Tip: If you know the exact incident time, enter it with ± minutes to narrow the data window. Otherwise, use a full date range to capture everything.

Search Criteria

Provide any combination — the tool will back-fill related devices and drivers automatically.

Vehicle name, serial number, or VIN. Auto-complete will suggest matches.
Driver first or last name. If omitted, drivers are discovered from trip data.
